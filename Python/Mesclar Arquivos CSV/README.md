# Readme - Mesclar Arquivos CSV

Este é um pequeno aplicativo em Python usando PyQt5 que permite selecionar uma pasta de origem contendo arquivos CSV e mesclá-los em um único arquivo CSV.

## Requisitos do Sistema

- Sistema Operacional: Windows

- Python 3.x instalado

- Bibliotecas: PyQt5

*Certifique-se de instalar as dependências antes de executar o aplicativo.*

pip install PyQt5

## Instruções de Uso

**Iniciar o Programa:**

- Execute o arquivo `mesclar.pyw` para iniciar o programa.

**Selecionar Pasta de Origem:**

- Clique no botão "Selecionar Pasta de Origem" e escolha a pasta que contém os arquivos CSV que deseja mesclar.

**Mesclar Arquivos CSV:**

- Após selecionar a pasta de origem, clique no botão "Mesclar Arquivos" para iniciar o processo de mesclagem.

- Um arquivo CSV mesclado será criado na pasta de origem, com o nome '- - - Arquivo Mesclado - - -.csv'.

**Verificar Resultados:**

- Após a conclusão da mesclagem, o programa imprimirá uma mensagem indicando que os arquivos foram mesclados com sucesso.

## Observações

- Certifique-se de selecionar uma pasta de origem contendo apenas arquivos CSV que deseja mesclar.

- O programa mesclará todos os arquivos CSV encontrados na pasta de origem.

- Caso encontre algum problema ou erro durante a execução, verifique se todas as dependências estão instaladas corretamente e se a pasta de origem contém arquivos CSV válidos.

- Aproveite a facilidade de uso e a eficiência do **Mesclar Arquivos CSV**!
